# Hey, I'm Sharven 👋

I'm an ML Engineer specializing in **computer vision** and **medical AI** — building production systems that sit at the intersection of deep learning and healthcare.

Currently at **Vetology Innovations** in Chicago, where I've deployed 50+ production models across classification, object detection, and segmentation on veterinary medical imaging. My work spans the full ML lifecycle — from self-supervised pretraining on 400K+ images to ONNX-optimized inference pipelines.

---

## 🔬 What I Work On

- **Medical Image Analysis** — segmentation, classification, and detection on radiographs and pathology slides
- **Self-Supervised Learning** — BYOL, SimCLR, MAE, DINOv2 for label-efficient medical imaging
- **Vision-Language Models** — CLIP, LLaVA, PaLiGemma for multimodal medical applications
- **Production ML** — PyTorch → MLflow → Airflow → ONNX/TensorRT on Azure + A100 GPUs
- **Foundation Models** — domain-specific pretraining and fine-tuning for specialized vision tasks
